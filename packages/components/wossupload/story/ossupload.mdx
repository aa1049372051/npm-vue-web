# 上传基本用法


```javascript
<template>
      <WOssUpload
        importType="Web"
        :my-file="{
          importType: '.png,.jpg',
          listType: 'text',
          fileCount: 1,
        }"
        :show-tips="false"
        :show-file-name="false"
      >
        <el-button link type="primary">上传</el-button>
      </WOssUpload>
</template>
```
